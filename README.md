## 🔬 three things I've shipped

<br/>

**🤖 HireAI** — *an agent that applies to jobs so you don't have to think about it*
An autonomous pipeline that reads a job description, decides which tools it needs, and writes a tailored cover letter + resume without me touching it. Function calling ties it to external APIs and document templates in real time.
`Python` `Claude API` `Function Calling` `Streamlit` → **80% less time spent drafting applications**

**🔎 Resume LLM Assistant** — *a second pair of eyes on your resume*
Feed it a resume and a job posting; it uses RAG over a skill-gap knowledge base and semantic matching to hand back 100+ specific, structured suggestions. What used to be a 15-minute manual pass is now under 2.
`Python` `Streamlit` `Claude API` `NLP` `RAG` → **100+ suggestions, sub-2-minute turnaround**

**🎪 Rotaract Club Website** — *proof that good UX moves real humans*
A mobile-first hub for a 200-member club — event calendars, galleries, testimonials, zero-downtime CI/CD deploys on Vercel. Not glamorous. Just effective.
`React` `Tailwind` `Vercel` → **40% jump in event turnout**
